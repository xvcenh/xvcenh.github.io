# xvcenh.github.io

我的个人博客，基于 GitHub Pages 和 Jekyll 构建。

## 快速开始

1. 在 `_posts/` 目录下创建 Markdown 文件
2. 推送更改，GitHub Pages 会自动构建

## 文章格式

```markdown
---
layout: post
title: "文章标题"
date: 2026-03-19 10:00:00 +0800
categories: [分类1, 分类2]
---

文章内容...
```

## 本地预览

```bash
bundle install
bundle exec jekyll serve
```
